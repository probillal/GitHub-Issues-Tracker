1️⃣ What is the difference between var, let, and const?
var: var is a keyword in JavaScript used to declare a variable. It has been in JavaScript since the beginning, before ES6 introduced let and const.
let: let is a keyword introduced in ES6 (2015) to declare variables in JavaScript. Unlike var, let is block-scoped.
const: const is a keyword introduced in ES6 (2015) to declare variables whose value cannot be reassigned after initialization. Like let, it is block-scoped and prevents re-declaration in the same scope.

2️⃣ What is the spread operator (...)?
Ans: The spread operator (...) in JavaScript is used to expand elements of an iterable (like an array or object) into individual elements. It is commonly used for copying, merging, or passing values.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map(): map() transforms each element and returns a new array.
filter(): filter() returns a new array with elements that match a condition. 
forEach(): forEach() simply loops through the array without returning a new array.

4️⃣ What is an arrow function?
Ans: An arrow function is a shorter syntax for writing functions in JavaScript. It was introduced in ES6 and uses the => (arrow) symbol. Arrow functions make code more concise and readable.

5️⃣ What are template literals?
Ans: “Template literals are strings defined using backticks (`) in JavaScript. They allow string interpolation using ${}, support multi-line strings, and make string formatting easier.”
